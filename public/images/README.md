# Images Folder

Place your product images in this folder:

- hero-image.jpg (main product hero image)
- product-1.jpg, product-2.jpg, etc. (product gallery images)
- kids-playing-1.jpg, kids-playing-2.jpg (about page images)

Then update the image paths in your components from:
`https://images.pexels.com/photos/...`

To:
`/images/your-image-name.jpg`
