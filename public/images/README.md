# Images Folder

Place your product images in this folder:

## Required Images for Your Website:

### Main Product Images:
- `hero-product.jpg` - Main hero section image (recommended: 800x600px)
- `classic-set.jpg` - Classic RockStack set
- `rainbow-set.jpg` - Colorful version
- `deluxe-set.jpg` - Premium set  
- `travel-set.jpg` - Travel-sized set

### About Page Images:
- `kids-playing-1.jpg` - Children playing with wooden toys
- `kids-building.jpg` - Child building structures
- `kids-social-play.jpg` - Kids playing together
- `focused-learning.jpg` - Child concentrating on play

### Gallery Images:
- `natural-wood-beauty.jpg` - Showcasing wood grain
- `endless-creativity.jpg` - Creative structures
- `perfect-balance.jpg` - Balanced stacking

### Contact Page:
- `customer-creation-1.jpg` through `customer-creation-4.jpg` - Customer photos

## Image Guidelines:
- Format: JPG or PNG
- Size: Optimize for web (under 500KB each)
- Dimensions: 800x600px or similar aspect ratio works well
- Quality: High enough to look professional

## After Adding Images:
1. Run `npm run build` to rebuild
2. Your images will be accessible at `/images/filename.jpg`
3. Deploy the new `out` folder contents
